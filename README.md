# PSFC Shift Calendar Sync

Instead of searching the Park Slope Food Coop (PSFC) [üìÖ Shift Calendar](https://members.foodcoop.com/services/shifts/) to sign up for an available shift, you can now just search in your favorite calendar app. As featured on the [`@foodcoop` Instagram](https://www.instagram.com/stories/highlights/18038560394618051/) and the [Linewaiter's Gazette](https://linewaitersgazette.com/2025/05/13/may-13-2025-letters/)!

This works by scraping the PSFC Shift Calendar every 20 minutes, reconciling with a public calendar.

## Getting Started

> [!NOTE]  
> You'll need to create an account on [foodcoop.news](https://foodcoop.news/signup?reason=integrations).

1. Navigate to https://foodcoop.news/integrations.
2. Select shifts (e.g. `üçû Bread Stocking`, `ü•¨ Produce Processing`) that you would like to sync with your calendar.
3. Click `Add iCal subscription` and select the application (e.g. Google Calendar, Outlook Calendar, etc.) to add your calendar.
